-------关于数组的方法-------

~~~
var a=[21,12,121,3]
1.arr.push() 在数组最后添加一项，会影响原数组，返回数组的长度
2.arr.pop() 不需要传值，删除最后一项，返回被删除的项，会影响原数组
3.unshift() 穿入需要新添加的项，向最前面添加，返回数组的长度，会影响原数组
4.shift() 不需要传值，删除数组的第一项，返回删除的项，会影响原数组
5.join() 按指定的内容将数组拼接成字符串返回，括号里是拼接的内容，若不传入参数，默认是",",如"+,-",对原数组不产生影响。
6.concat()   将传入的两个数组拼接起来，不影响原数组
7.indexOf()  从开头开始查找指定项在数组第一次出现的地方，若找不到返回-1，
8.lastindexOf() 从数组最后开始找
9.reverse() 不需要传入参数，翻转数组。会影响原数组
10.slice() 需要传入两个参数，即开始截取的地方和结束的地方，从原数组截取项，，不影响原数组。
11.splice() 可以从数组中删除，插入，替换，返回的值永远都是被删除的项，若没有删除的项，则返回空数组，会影响原数组。
	一、删除
		可以删除任意数量的项，需要指定两个参数，要删除的第一项的位置和要删除的项数a
	二、插入 
		可以向指定位置插入任意数量的项，只需提供3个参数，起始位置，0(要删除的项数)、要插入的项，可以传入多个参数，splice(2.0,"blue","black"),会从当前数组的位置2开始，插入字符串"blue"和"black",会改变原数组
	三、替换
		可以向数组插入任意数量的项，并且同时删除任意数量的项，只需要指定3个参数：起始参数、起始位置、要删除的项数、要插入的任意数量的项、要插入的项数不必等于删除的项数
		splice(2,1,"red","blue"）
		项数指的都是下标
12 forEach() 是一个操作，括号中传的的是参数，无返回值，会影响原数组
	forEach(function(ele,ind, arr){
        ind:每一项的下标
        ele：每一项的内容
	})
13 sort() 会影响原数组，返回排序后的数组
	在括号中传入函数function(a,b){
        return a-b;
	}
14 map() 映射，一定要return ，不会改变原数组，进行一定的函数方法处理后返回
15 filter()返回满足条件的项组成数组
16 reduce() 从左往右缩减数组
  var result=arr.reduce(function(a,b){
      return a+b;
      return a+(b-1);
      return a*b
  });
 17. reduceRight() 按规则从左往右缩减数组
 18 every() 检测每一个项，如果都满足条件，则返回true否则返回false
19 some() 检测每一个项，如果存在满足条件，则返回true否则false

~~~

--------字符串方法-------

~~~
1.length 返回字符串的长度
	var str=“accc的的的123”;
	console.log(str.length)
2.indexOf("a") 返回的是字符串中检索的指定字符的第一次出现的位置	若给出两个值，第二个值是从指定的下标开始查找
	console.log(str.indexOf("的"));//4
	console.log(str.indexOf("的"，7))；//-1
3.lastIndexOf() 返回字符串中检索的字符最后一次出现的位置
 console.log(str.lastIndexOf("a"))//0
4. substring(开始的下标，结束的下标) 截取字符串
	var a=str.substring(1,4);//下标从1开始，到3结束，不包括下标4的内容
	console.log(a)//acc
console.log(str) //不会影响原先字符串
5.slice(开始的下标，结束得到的下标(可以为负值)) 提取字符串的片段，并在新的字符串中返回被提取的部分
  	var a=str.slice(0,4);
  	cosole.log(a)//aacc
  	var a=str.slice(0,-5)
  	cosnole.log(a) //aacc的，第二个值会用长度减去它，然后截取
  	console.log(str) //不影响原字符串
6. substr(开始的下标，要截取的个数)从起始索引号提取字符串中指定数目的字符
	console.log(str.substr(5,5))//的的123 若只给一个值，那么和这样得到的1结果相同
	7.分割字符串
	string.split(",")//按照逗号分割字符串


~~~
